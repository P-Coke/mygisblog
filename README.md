# zqlovegis Theme

A custom WordPress block theme for `zqlovegis.cn`.

## Positioning

This theme is designed for a modern GIS-focused portfolio and technical blog. It emphasizes:

- Full-stack engineering
- Python for GIS
- Spatial data pipelines
- Cartographic design
- Project storytelling

## Local usage

Copy this repository into:

`wp-content/themes/zqlovegis-theme`

Then activate the theme in the WordPress admin panel.

## Suggested workflow

1. Develop the theme locally.
2. Commit and push to GitHub.
3. On the server, pull the latest changes into the theme directory.

## Theme structure

- `style.css`: theme metadata and shared styles
- `theme.json`: block theme settings and design tokens
- `functions.php`: theme setup helpers
- `templates/`: page templates
- `parts/`: reusable template parts
